# Superpowers Planning Archive

This directory stores specs and execution plans produced during prior Superpowers-driven development passes.

Status rules:

- `plans/` is an execution-history archive by default.
- `specs/` may be used as background design material when a current source-of-truth document links to it.
- Historical files may contain old project names, paths, package identities, commands, and branch names.
- Do not execute a plan from this directory directly without first checking `docs/README.md`, `docs/STATUS.md`, and the current code layout.
