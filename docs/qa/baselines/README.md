# Playback Quality Baselines

This directory stores committed playback-quality baseline/report-set artifacts.

Status rules:

- Treat these files as frozen evidence for their recorded eval version.
- Do not edit JSON reports manually to make a result pass.
- Refresh a baseline only through the documented quality-run commands and commit the command context in `docs/STATUS.md`.
- Keep private/local report-sets under ignored `docs/qa/private/` paths, not here.
- During documentation cleanup, do not rewrite these artifacts for branding, path, or formatting-only changes.
